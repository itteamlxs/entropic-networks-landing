# Entropic Networks Landing Page

Landing page de Entropic Networks, diseñada para presentar de forma clara y profesional nuestros servicios tecnológicos. Esta estructura base está desarrollada en **PHP**, **Tailwind CSS** y **JavaScript minimalista**, lista para integrar contenido real y desplegar de forma segura.

## Quiénes Somos

En Entropic Networks, nos especializamos en brindar una gama integral de servicios tecnológicos diseñados para garantizar el óptimo funcionamiento de tus equipos y sistemas. Nuestro propósito es transformar los desafíos tecnológicos en soluciones efectivas que impulsen la productividad y protejan tu inversión.

Nuestro enfoque integral asegura que cada aspecto de tu infraestructura tecnológica esté respaldado por soporte profesional, soluciones efectivas y un compromiso constante con la excelencia.

> "La entropía es caos. Entropic Networks es control."

## Qué Ofrecemos

- **Mantenimiento Especializado**: Optimización de hardware y software para prolongar la vida útil de tus PCs y laptops, asegurando rendimiento óptimo.  
- **Instalación y Configuración de Redes**: Soluciones de conectividad robustas y eficientes, adaptadas a tus necesidades específicas.  
- **Ciberseguridad**: Protección de datos y sistemas mediante soluciones innovadoras que anticipan amenazas y reducen vulnerabilidades.  
- **Consultoría Personalizada**: Asesoramiento estratégico para optimizar el rendimiento de tus sistemas, mejorar la seguridad y alinear la tecnología con los objetivos de tu empresa.

## Tecnologías

- **PHP**: Estructura base y manejo de plantillas.  
- **Tailwind CSS**: Diseño responsive y moderno.  
- **JavaScript minimalista**: Interacciones y microanimaciones.  

## Despliegue y Desarrollo

- Desarrollo local con Tailwind y PHP.  
- Gestión de versiones y CI/CD mediante Git y GitHub.  
- Posibilidad de despliegue directo en servidor Linux (Ubuntu 24.04 LTS) mediante `git pull` y build de Tailwind.
